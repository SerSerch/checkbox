# checkbox

* [Установить Node.js LTS версию](https://nodejs.org/ru/)

* в редакторе кода открыть Terminal и выполнить команды

* что бы установить все модули
``` bash
npm i
```

* что бы запустить локально сервер
``` bash
npm start
```

* что бы сделать сборку в папку dist
``` bash
npm run build
```
